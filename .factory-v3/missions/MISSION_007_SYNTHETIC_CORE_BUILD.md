## Dependency Policy
No package installation is approved.

Approved runtime:
- `python3` standard library only.

Approved dependency basis:
- Python standard library is already available locally and requires no package install.
- JSON fixtures use standard `json`.
- Tests use standard `unittest`.
- Dataclasses and typing use standard library modules.

Forbidden during this mission:
- `pip install`, `npm install`, `brew install`, or any package manager.
- Garmin, Apple Health, Google Health/Fitbit, Health Connect, Polar, Strava, Telegram, OCR/vision, voice transcription, real medical PDF, scheduler, notification, queue, worker, daemon, Hermes, or deployment dependency.

Rollback plan:
- If the stdlib-only scaffold cannot satisfy the gates, stop and record the missing V3/app capability. Do not install packages.
